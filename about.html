Bootstrap helps a lot in building responsive layouts quickly, but it is not enough on its own. You still need custom CSS to match exact designs, handle unique UI requirements, and customize themes. Bootstrap components often need overrides for spacing, colors, and behavior. Real-world projects always combine Bootstrap classes with custom CSS. Interviewers expect you to know Bootstrap and core CSS, not just utility classes.
